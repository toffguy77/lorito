> **Суть**
> Расширение перифраз A2 (см. A2: перифразы). Группируем по типу: фаза действия, длительность, повтор, приближение.

## ⏱️ С инфинитивом

| Перифраза | значение | пример |
|-----------|----------|--------|
| **ir a + inf** | намерение | *Voy a salir.* |
| **acabar de + inf** | только что | *Acabo de llegar.* |
| **volver a + inf** | снова | *Volvió a llamar.* |
| **empezar/ponerse a + inf** | начало | *Se puso a llorar.* |
| **dejar de + inf** | прекращение | *Dejé de fumar.* |
| **soler + inf** | привычка | *Suelo madrugar.* |
| **tener que / deber + inf** | долженствование | *Tienes que ir.* (см. долженствование) |

## 🔄 С герундием (длительность/процесс)

| Перифраза | значение | пример |
|-----------|----------|--------|
| **estar + gerundio** | действие сейчас | *Estoy trabajando.* |
| **llevar + время + gerundio** | длится уже… | *Llevo dos horas estudiando.* |
| **seguir / continuar + gerundio** | продолжать | *Sigue lloviendo.* |
| **ir + gerundio** | постепенно | *Va mejorando poco a poco.* |
| **andar + gerundio** | «ходит и делает» | *Anda buscando trabajo.* |

## ✅ С причастием (результат)
- **llevar + participio:** *Llevo escritas tres páginas.* — Написал уже три страницы.
- **tener + participio:** *Tengo hecha la maleta.* — Чемодан уже собран.

## 🎯 Примеры в речи
- *Llevo un año **aprendiendo** español y sigo **mejorando**.*
- *Acababa de **salir** cuando empezó a **llover**.*
