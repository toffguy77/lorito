> **Суть**
> На C2 нужно выражать тонкие эмоциональные и абстрактные оттенки точно. Это область, где богатство словаря особенно заметно.

## 😊😣 Градация эмоций
| Слабее → сильнее |
|-------------------|
| *contento → alegre → dichoso → eufórico → exultante* |
| *molesto → enfadado → furioso → iracundo → encolerizado* |
| *triste → apenado → abatido → desolado → consternado* |
| *preocupado → inquieto → angustiado → atormentado* |
| *sorprendido → asombrado → atónito → estupefacto → pasmado* |

## 🎚️ Тонкие эмоциональные понятия (трудно перевести одним словом)
- *añoranza / nostalgia* (тоска по прошлому/дому), *desazón* (смутная тревога), *sosiego* (умиротворение), *recelo* (опасливое недоверие), *hastío* (пресыщение/скука), *anhelo* (страстное желание), *congoja* (тоска, удушающая печаль), *zozobra* (тревожная неопределённость).

## 🧠 Абстрактные существительные от прилагательных/глаголов
Словообразование (см. словообразование):
- *-dad/-ez/-eza/-ura/-itud:* *honestidad, sensatez, nobleza, cordura, exactitud*.
- *-miento/-ción:* *resentimiento, abnegación*.

## 🗣️ Глаголы психических состояний с régimen
- *anhelar algo, recelar de, regodearse en, enorgullecerse de, avergonzarse de, conformarse con* (см. управление предлогами).

## 🎨 Образная передача чувств
- *un nudo en la garganta, se le cayó el alma a los pies, le hervía la sangre, sentir mariposas en el estómago* (см. образность).

## 🎯 Совет
Заведите тематические «гнёзда» эмоций с градацией интенсивности и коннотацией (см. синонимы и коннотация); это резко повышает выразительность.
