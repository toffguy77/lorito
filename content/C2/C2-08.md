> **Суть**
> На C2 «синонимы» различаются микро-нюансами употребления, сочетаемости, интенсивности. Разбор частотных трудных пар. Опора на C1 (см. C1: синонимы).

## ⚖️ Глаголы — близкие, но разные
| Пара | различие |
|------|----------|
| *acordar / recordar / acordarse* | договориться / помнить / вспоминать |
| *darse cuenta / percatarse / reparar en* | осознать (нейтр./книжн./обратить внимание) |
| *lograr / conseguir / alcanzar* | добиться (усилие / получить / достичь предела) |
| *demostrar / probar / evidenciar* | доказать (показать / подтвердить / выявить) |
| *deber / tener que / haber de* | долг (моральный / необходимость / книжн.) |

## 🎚️ Существительные — оттенки
| Пара | различие |
|------|----------|
| *error / equivocación / fallo / errata* | ошибка / оплошность / сбой / опечатка |
| *problema / cuestión / asunto / inconveniente* | проблема / вопрос / дело / помеха |
| *miedo / temor / pavor / recelo* | страх по нарастающей / + недоверие |
| *trabajo / labor / faena / tarea* | оттенки труда/задачи |

## 🎨 Прилагательные — интенсивность/коннотация
- *raro / extraño / peculiar / estrafalario* (странный → эксцентричный).
- *viejo / anciano / añejo / vetusto* (старый: человек / вино / ветхий).
- *grande / enorme / colosal / descomunal*.

## 🧩 Различие по сочетаемости (колокации)
Даже близкие слова требуют разных партнёров:
- *cometer un error* (не *hacer*), *zanjar una discusión* (не *terminar* в этом смысле).
(см. колокации)

## 🎯 Метод
Сверяйте в **толковом словаре** (DRAE) определение + примеры + пометы (*coloq., desus., cult.*); смотрите типичные сочетания в корпусе (CORPES).
