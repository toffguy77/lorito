> **Суть**
> На C2 «синонимы» различаются микро-нюансами: значением, сочетаемостью, интенсивностью, регистром. Полных синонимов почти не бывает — каждое слово несёт свой оттенок. Разбор частотных трудных пар. Опора на C1 (см. C1: синонимы).

## ⚖️ Глаголы — близкие, но разные
| Пара | различие |
|------|----------|
| *acordar / recordar / acordarse* | договориться о чём-то / помнить (книжн., перех.) / вспоминать (*acordarse de*) |
| *darse cuenta / percatarse / reparar en* | осознать (нейтр.) / заметить (книжн.) / обратить внимание на деталь |
| *lograr / conseguir / alcanzar* | добиться усилием / получить, раздобыть / достичь предела, цели |
| *demostrar / probar / evidenciar* | показать логикой / подтвердить фактом / делать очевидным |
| *deber / tener que / haber de* | моральный долг / внешняя необходимость / книжное долженствование |
| *oír / escuchar* | слышать (пассивно) / слушать (активно, со вниманием) |
| *saber / conocer* | знать факт, уметь / быть знакомым с человеком, местом |

## 🎚️ Существительные — оттенки
| Пара | различие |
|------|----------|
| *error / equivocación / fallo / errata* | ошибка вообще / оплошность по невнимательности / технический сбой / опечатка в тексте |
| *problema / cuestión / asunto / inconveniente* | проблема / вопрос для обсуждения / дело / мелкая помеха |
| *miedo / temor / pavor / recelo* | страх по нарастающей (*miedo < temor < pavor*) / + недоверие, опаска (*recelo*) |
| *trabajo / labor / faena / tarea* | труд вообще / труд как миссия / конкретная физ. работа / поставленное задание |

## 🎨 Прилагательные — интенсивность и коннотация
Близкие прилагательные различаются силой признака и сочетаемостью:

- *raro → extraño → peculiar → estrafalario* (странный → эксцентричный, по нарастанию необычности).
- *viejo / anciano / añejo / vetusto* (старый: вещь и человек / только о человеке, уважительно / о вине, выдержанный / ветхий, обветшалый).
- *grande → enorme → colosal → descomunal* (большой → исполинский).
- *bonito → hermoso → precioso → sublime* (красивый → возвышенный).

## 🧩 Различие по сочетаемости (колокации)
Даже очень близкие слова требуют разных «партнёров»:

- *cometer un error* (не *hacer un error*);
- *zanjar una discusión / una polémica* (решительно прекратить, не просто *terminar*);
- *infligir un daño*, но *causar un daño*;
- *acuciante necesidad*, но не *acuciante problema* без причины.

## ⚠️ Частые ошибки
- **«Один русский = одно испанское».** Русское «знать» — это и *saber*, и *conocer*; путаница даёт *conozco la respuesta* вместо *sé la respuesta*. Различайте по типу объекта.
- **Игнор регистра у синонима.** *Pavor* и *miedo* не взаимозаменяемы: *pavor* — сильный, литературный. Подбирайте оттенок под контекст.
- **Нарушение колокации.** *Hacer un error*, *terminar una discusión* в смысле «решительно закрыть» — звучит не по-испански. Запоминайте слово вместе с его глаголом.
- **Опора только на словарь синонимов.** Список синонимов не показывает нюанс. Сверяйте определение, примеры и пометы (*coloq., desus., cult.*) в **толковом словаре** (DRAE) и типичные сочетания в корпусе (CORPES).
