> **Суть**
> На C2 ценится **точное слово** (*mot juste*) вместо описательных оборотов. Один меткий термин лучше длинного перифраза. Цель — **плотность** текста: максимум смысла при минимуме слов, без потери ясности.

## 🔧 Описательный оборот → одно слово
Замена перифраза (описания в несколько слов) на единый точный термин делает речь собраннее и взрослее:

| Перифраз | Точное слово |
|----------|--------------|
| *la persona que conduce el autobús* | *el conductor* |
| *hacer que algo sea más fácil* | *facilitar* |
| *dar algo de nuevo a su dueño* | *devolver / restituir* |
| *que no se puede evitar* | *inevitable / ineludible* |
| *poner fin a algo* | *concluir / zanjar* |
| *que ocurre cada año* | *anual* |
| *falta de interés o de ganas* | *desidia / apatía* |
| *que no se puede expresar con palabras* | *inefable* |
| *que sucede después de la muerte* | *póstumo* |
| *que dura muy poco* | *efímero / fugaz* |

## ✂️ Экономия (concisión)
Убираем лишние слова без потери смысла — частая правка академического и делового текста:

- *en el día de hoy* → ***hoy***
- *a pesar del hecho de que* → ***aunque / pese a que***
- *tiene la capacidad de* → ***puede***
- *en relación con el tema de* → ***sobre / acerca de***
- *de color rojo* → ***rojo***
- *con el fin de* → ***para***
- *en el caso de que* → ***si***
- *hacer uso de* → ***usar / emplear***
- *llevar a cabo* → ***realizar / efectuar***

## 🪓 Избегать «relleno» (наполнителей)
Слова-пустышки ослабляют текст и звучат по-детски: *cosa, algo, hacer, tema, aspecto, tipo de, un poco*. Заменяйте их конкретикой:

- ❌ *Hizo una cosa importante.* → ✅ *Logró un avance decisivo.*
- ❌ *Es un tema complicado.* → ✅ *Es una cuestión espinosa.*
- ❌ *Tiene un aspecto raro.* → ✅ *Tiene un aire extraño.*
- ❌ *Dijo algo sobre el plan.* → ✅ *Aludió al plan.*

## 🧩 Точные глаголы вместо «hacer / poner / dar»
Глаголы широкого значения почти всегда имеют более меткую замену под контекст:

- *hacer una pregunta* → ***formular*** *una pregunta*
- *hacer un proyecto* → ***elaborar*** *un proyecto*
- *poner una norma* → ***establecer*** *una norma*
- *dar una explicación* → ***ofrecer / aducir*** *una explicación*
- *hacer una ley* → ***promulgar*** *una ley*
- *dar un problema* → ***plantear*** *un problema*

## ⚠️ Частые ошибки
- **Калькирование русских перифразов.** Русское «является» порождает тяжёлые конструкции: *es algo que tiene la función de…* вместо простого глагола. Спрашивайте себя: «каким одним словом это сказать?».
- **Точность ≠ редкие слова ради вычурности.** Нагромождение книжных слов (*ósculo* вместо *beso*) звучит *cursi / pedante*. Цель — **ясность + плотность**, а не демонстрация словаря.
- **Ложная экономия за счёт смысла.** Нельзя выкидывать слова, несущие нюанс: *casi inevitable* ≠ *inevitable*. Сокращайте только балласт.
- **Регистр важнее краткости.** *Palmar* короче, чем *fallecer*, но в некрологе уместно второе. Уместность регистра — приоритет (см. стилистическая правка).
