> **Суть**
> На C2 ценится **точное слово** (mot juste) вместо описательных оборотов. Один меткий термин лучше длинного перифраза.

## 🔧 Описательный оборот → одно слово
| Перифраз | Точное слово |
|----------|--------------|
| *la persona que conduce el autobús* | *el conductor* |
| *hacer que algo sea más fácil* | *facilitar* |
| *dar de nuevo* | *devolver / restituir* |
| *que no se puede evitar* | *inevitable / ineludible* |
| *poner fin a algo* | *concluir / zanjar* |
| *que ocurre cada año* | *anual* |
| *falta de interés* | *desidia / apatía* |

## ✂️ Экономия (concisión)
Убираем лишнее без потери смысла:
- *en el día de hoy* → ***hoy***
- *a pesar del hecho de que* → ***aunque / pese a que***
- *tiene la capacidad de* → ***puede***
- *en relación con el tema de* → ***sobre / acerca de***
- *de color rojo* → ***rojo***

## 🪓 Избегать «relleno» (наполнителей)
Слова-пустышки ослабляют текст: *cosa, algo, hacer, tema, aspecto, tipo de*.
- ❌ *Hizo una cosa importante.* → ✅ *Logró un avance decisivo.*
- ❌ *Es un tema complicado.* → ✅ *Es una cuestión espinosa.*

## 🧩 Точные глаголы вместо «hacer/poner/dar»
- *hacer una pregunta → **formular** una pregunta*; *hacer un proyecto → **elaborar***; *poner una norma → **establecer***; *dar una explicación → **ofrecer/aducir***.
(см. колокации: колокации)

## ⚠️ Баланс
Точность ≠ редкие слова ради вычурности. Цель — **ясность + плотность**, а не демонстрация словаря. Уместность регистра важнее (см. стилистическая правка).
