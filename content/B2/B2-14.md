> **Суть**
> *se* — самое многозначное слово испанского. На B2 нужно уверенно различать его шесть основных функций и не путать пассивную и безличную конструкции.

## 📐 Шесть значений se

| Значение | смысл | пример |
|----------|-------|--------|
| **1. Возвратное (reflexivo)** | действие на себя | *Se lava.* — Моется. |
| **2. Взаимное (recíproco)** | друг друга | *Se quieren.* — Любят друг друга. |
| **3. Pasiva refleja** | пассив (есть объект-подлежащее) | *Se venden pisos.* — Продаются квартиры. |
| **4. Безличное (impersonal)** | без деятеля, 3 л. ед. | *Se vive bien aquí.* — Здесь хорошо живётся. |
| **5. Дативное «se» (le→se)** | замена le/les перед lo/la | *Se lo dije.* — Я ему это сказал. |
| **6. Аспектуальное/valorativo** | оттенок завершённости/эмоции | *Se comió todo el pastel.* — Съел весь торт. |

## 🔁 Возвратное и взаимное
Возвратное *se* — действие субъекта на самого себя; местоимение меняется по лицам (*me, te, se, nos, os, se*):
- *Me **lavo** las manos. ¿**Te** has duchado ya?*

Взаимное *se* возможно только во множественном числе («друг друга»). Для снятия двусмысленности добавляют уточнение:
- *Se escriben (el uno al otro).* — Переписываются друг с другом.
- *Se ayudan mutuamente.*

## 📦 Pasiva refleja vs безличное se
Это разные конструкции — их легко спутать.

- **Pasiva refleja:** есть грамматическое подлежащее (вещь), глагол **согласуется** с ним в числе:
  - *Se vende **un piso**.* (ед.) → *Se vend**en** **pisos**.* (мн.)
  - *Aquí se habla**n** varios idiomas.*
- **Безличное se:** деятеля нет вообще, глагол **всегда** в 3 л. **ед.ч.** (с непереходными глаголами или с предлогом *a* + одушевлённый объект):
  - *Aquí se vive bien.* / *Se trabaja mucho.*
  - *Se busca **a** los culpables.* (не *se buscan*: есть *a* перед лицом → безличное).

## 🍰 Аспектуальное se (важно для B2)
Подчёркивает полноту/вовлечённость; объект обычно «весь/конкретный» (с артиклем или *todo*):
- *Comió pan* (ел хлеб) → ***Se** comió **todo** el pan.* (съел весь).
- *Bebió* → ***Se** bebió la botella entera.*
- *Leyó el libro* → ***Se** leyó el libro en una noche.* (прочёл залпом).
- *Fue* → ***Se** fue.* (ушёл — оттенок «убрался»). Также: *dormirse, morirse, llevarse, saberse, leerse, conocerse*.

## 🔄 Глаголы, меняющие смысл с se
| без se | с se |
|--------|------|
| ir (идти) | irse (уходить) |
| dormir (спать) | dormirse (засыпать) |
| acordar (договориться) | acordarse de (помнить) |
| quedar (договориться о встрече) | quedarse (оставаться) |
| parecer (казаться) | parecerse a (быть похожим) |
| ocurrir (происходить) | ocurrirse (прийти в голову) |

## 🧷 «se» непроизвольности (se me…)
Снимает ответственность («оно само»); порядок — *se + me/te/le… + глагол*, глагол согласуется с предметом:
- ***Se me** rompió el vaso.* — У меня разбился стакан (не «я разбил»).
- ***Se me** olvidó.* — Я забыл (само забылось). / ***Se nos** olvidaron las llaves.*
- ***Se le** cayó el teléfono.* — У него упал телефон.

## ⚠️ Частые ошибки
- **Согласование в pasiva refleja.** По-русски «продают квартиры», но в испанском подлежащее — *pisos*, поэтому ***Se venden** pisos*, не *Se vende pisos*.
- **Безличное всегда в ед.ч.** *Se busca **a** los responsables* (не *se buscan*): предлог *a* перед лицом сигнализирует безличную конструкцию.
- **Путать pasiva refleja и impersonal.** Если за глаголом стоит предмет-подлежащее → согласуем (*se necesit**an** camareros*). Если объект одушевлён с *a* или глагол непереходный → ед.ч.
- **«se me olvidó», а не «yo olvidé».** Для оттенка «само забылось / случайно» нужна конструкция непроизвольности с *se*; забыли поставить *se* — теряется смысл.
- **Порядок клитик.** В *se me cayó* сначала *se*, потом *me*; обратный порядок (*me se*) — грубая ошибка, типичная у изучающих.
