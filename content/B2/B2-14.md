> **Суть**
> *se* — самое многозначное слово испанского. На B2 нужно различать его 6 основных функций.

## 📐 Шесть значений se

| Значение | смысл | пример |
|----------|-------|--------|
| **1. Возвратное (reflexivo)** | действие на себя | *Se lava.* — Моется. |
| **2. Взаимное (recíproco)** | друг друга | *Se quieren.* — Любят друг друга. |
| **3. Pasiva refleja** | пассив (есть объект-подлежащее) | *Se venden pisos.* |
| **4. Безличное (impersonal)** | без деятеля, 3 л. ед. | *Se vive bien aquí.* |
| **5. Дативное «se» (le→se)** | замена le/les перед lo/la | *Se lo dije.* (см. se lo) |
| **6. Аспектуальное/valorativo** | оттенок завершённости/эмоции | *Se comió todo el pastel.* |

## 🍰 Аспектуальное se (важно для B2)
Подчёркивает полноту/вовлечённость, объект обычно «весь/конкретный»:
- *Comió pan* (ел хлеб) → ***Se** comió **todo** el pan.* (съел весь).
- *Bebió* → ***Se** bebió la botella entera.*
- *Fue* → ***Se** fue* (ушёл — оттенок «убрался»). Также: *dormirse, morirse, llevarse, saberse, leerse*.

## 🔄 Глаголы, меняющие смысл с se
| без se | с se |
|--------|------|
| ir (идти) | irse (уходить) |
| dormir (спать) | dormirse (засыпать) |
| acordar (договориться) | acordarse de (помнить) |
| quedar (договориться о встрече) | quedarse (оставаться) |
| parecer (казаться) | parecerse a (быть похожим) |

## 🧷 «se» непроизвольности (se me…)
Снимает ответственность («оно само»):
- ***Se me** rompió el vaso.* — У меня разбился стакан (не «я разбил»).
- ***Se me** olvidó.* — Я забыл (само забылось).
