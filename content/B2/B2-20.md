> **Суть**
> Понимание словообразовательных моделей резко расширяет словарный запас: по знакомому корню можно угадать новые слова.

## 🔧 Суффиксы существительных
| Суффикс | значение | пример |
|---------|----------|--------|
| -ción/-sión | действие/результат | *organización, decisión* |
| -dad/-tad | качество | *libertad, realidad* |
| -miento | процесс | *crecimiento, movimiento* |
| -eza/-ura | свойство | *belleza, altura* |
| -aje | собирательное | *aprendizaje, kilometraje* |
| -ería | место/занятие | *panadería, cafetería* |
| -ista/-or(a) | деятель | *artista, escritor* |

## 🎨 Суффиксы прилагательных
*-oso (cariñoso), -able/-ible (amable, posible), -al (cultural), -ivo (creativo), -ante (importante).*

## 🔁 Глагольные суффиксы
*-izar (organizar), -ificar (clarificar), -ear (telefonear), -ecer (oscurecer).*

## ➕ Префиксы
| Префикс | значение | пример |
|---------|----------|--------|
| in-/im-/i- | отрицание | *imposible, irreal, ilegal* |
| des- | обратное действие | *deshacer, desorden* |
| re- | повтор/усиление | *rehacer, requetebueno* |
| pre-/pos(t)- | до/после | *prever, posguerra* |
| sub-/super- | под/сверх | *subir, supermercado* |
| anti-/contra- | против | *antivirus, contraataque* |

## 🐤 Diminutivos и aumentativos
- **Уменьшительные:** -ito/a, -illo/a, -cito → *casita, momentito, panecillo* (нежность, малость).
- **Увеличительные:** -ón/-ona, -azo, -ote → *grandón, golpazo* (большое, удар, грубость).
- **Уничижительные:** -ucho, -ajo → *casucha, pequeñajo*.

> *-azo* также = «удар чем-то»: *codazo* (удар локтем), *flechazo* (любовь с первого взгляда).

## 🎯 Польза
Зная *pan → panadería → panadero*, выводите целые гнёзда слов.
