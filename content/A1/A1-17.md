> **Суть**
> **hay** = «есть / имеется / находится» (безличная форма глагола *haber*). Говорит о **существовании** чего-либо. Форма **одна** для ед. и мн. числа.

## 📝 Употребление
- *En la mesa **hay** un libro.* — На столе есть книга.
- ***Hay** tres sillas.* — Есть три стула.
- *¿**Hay** un banco cerca?* — Здесь рядом есть банк?

## ⚖️ hay vs estar — важное различие!

| | **hay** | **estar** |
|--|---------|-----------|
| Что выражает | существование (новое, неопределённое) | местоположение известного |
| Артикль | неопределённый / число / без артикля | определённый / имя собственное |
| Пример | *Hay **un** museo aquí.* | *El museo **está** aquí.* |
| Пример | *Hay **muchos** coches.* | *Los coches **están** en la calle.* |

**Правило:** известный объект (с *el/la*, имя) → **estar**; новый/неопределённый → **hay**.

## ⚠️ Частые ошибки
- ❌ *Hay el libro* → ✅ *Hay un libro* / *El libro está...*
- *hay* не меняется по числу: ❌ *hayn* — такого нет, всегда *hay*.
