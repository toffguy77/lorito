> **Суть**
> **hay** = «есть / имеется / находится» (безличная форма глагола *haber*). Говорит о **существовании / наличии** чего-либо. Форма **одна** для единственного и множественного числа — никогда не меняется.

## 📝 Употребление
- *En la mesa **hay** un libro.* — На столе есть книга.
- ***Hay** tres sillas.* — Есть три стула.
- *¿**Hay** un banco cerca?* — Здесь рядом есть банк?
- *En la clase **hay** muchos estudiantes.* — В классе много студентов.
- *No **hay** leche.* — Нет молока.

## 🔢 Что ставится после hay
После *hay* идёт **неопределённый** объект: с артиклем *un/una/unos/unas*, с числом, с *mucho/poco*, или вообще без артикля (с неисчисляемыми и во мн. ч.):
- *Hay **un** problema.* / *Hay **unas** flores.*
- *Hay **dos** ventanas.*
- *Hay **mucha** gente.*
- *Hay agua.* (без артикля — неисчисляемое)

## ⚖️ hay vs estar — важное различие!

| | **hay** | **estar** |
|--|---------|-----------|
| Что выражает | существование (новое, неопределённое) | местоположение известного |
| Артикль | неопределённый / число / без артикля | определённый (*el/la*) / имя собственное |
| Пример | *Hay **un** museo aquí.* | *El museo **está** aquí.* |
| Пример | *Hay **muchos** coches.* | *Los coches **están** en la calle.* |

**Правило:** известный, конкретный объект (с *el/la*, имя собственное, *mi/tu*) → **estar**; новый, неопределённый, впервые упомянутый → **hay**.
- *¿Dónde **está** el baño?* (конкретный, известный) vs *¿**Hay** un baño aquí?* (существует ли вообще)

## ⚠️ Частые ошибки
- После *hay* **нельзя определённый артикль**: ❌ *Hay el libro* → ✅ *Hay un libro* (есть книга) / *El libro está...* (книга находится).
- *hay* **не меняется по числу**: ❌ *hayn*, ❌ *hays* — таких форм нет, всегда *hay* и для одного, и для многих: *Hay un coche / Hay diez coches.*
- Не путайте с *estar*: «На столе **есть** книга» (наличие) = *hay*; «Книга **находится** на столе» (конкретная) = *está*.
