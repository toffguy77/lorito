> **Суть**
> Три источника лексических ловушек: многозначность (полисемия), близкие по форме слова (паронимы) и обманчивое сходство с русским/английским (ложные друзья). Различать их — задача именно уровня C1.

## 🔀 Полисемия (одно слово — много значений)
Одно слово покрывает несколько значений; нужное снимается контекстом и колокацией:

| Слово | значения |
|-------|----------|
| **banco** | скамейка / банк / косяк рыб (*banco de peces*) |
| **planta** | растение / этаж (*primera planta*) / завод / ступня (*planta del pie*) |
| **cabo** | мыс / капрал / конец (*al cabo de un año* — спустя год) |
| **dejar** | оставлять / позволять (*déjame ir*) / бросать (*dejar de fumar*) |
| **pasar** | проходить / случаться (*¿qué pasa?*) / передавать (*pásame la sal*) |
| **tiempo** | время / погода (*hace buen tiempo*) / грам. время |
| **cura** | лечение (la cura) / священник (el cura — меняет род!) |

Контекст и колокация снимают неоднозначность (см. колокации). Иногда значение различает **род**: *el cura* (священник) vs *la cura* (лечение); *el capital* (капитал) vs *la capital* (столица).

## ⚖️ Паронимы (похожи по форме — разный смысл)
Различаются одной-двумя буквами; ошибка меняет смысл фразы:

| Пара | различие |
|------|----------|
| *actual* (нынешний) / *actuar* (действовать) | |
| *especie* (вид, биол.) / *especia* (специя) | |
| *infligir* (наносить — ущерб, наказание) / *infringir* (нарушать — закон) | |
| *absolver* (оправдать) / *absorber* (впитывать) | |
| *aptitud* (способность) / *actitud* (отношение, настрой) | |
| *prever* (предвидеть) / *proveer* (снабжать) | |
| *adoptar* (принять, усыновить) / *adaptar* (приспособить) | |
| *eminente* (выдающийся) / *inminente* (неминуемый) | |

## 🎭 Ложные друзья (с русским/английским)
Похожи по форме на русское/английское слово, но значат другое:

| Испанское | НЕ значит | значит |
|-----------|-----------|--------|
| *éxito* | «выход» | успех (выход — *salida*) |
| *fábrica* | «фабрика тканей» | завод (любой) |
| *carta* | «карта» | письмо / меню (карта — *mapa / tarjeta*) |
| *embarazada* | «смущённая» | беременная (смущённая — *avergonzada*) |
| *sensible* | англ. «sensible» (разумный) | чувствительный (разумный — *sensato*) |
| *actualmente* | англ. «actually» | в настоящее время (на самом деле — *en realidad*) |
| *realizar* | англ. «realize» (осознать) | осуществлять (осознать — *darse cuenta*) |
| *ropa* | рус. «ропа?» / англ. «rope» | одежда (верёвка — *cuerda*) |
| *constipado* | «constipated» | простуженный (запор — *estreñido*) |
| *largo* | англ. «large» | длинный (большой — *grande*) |
| *éxito* / *suceso* | | *suceso* = происшествие, НЕ «успех» |

## ⚠️ Частые ошибки
Самые заметные ошибки русскоязычных — именно ложные друзья:
- ❌ *Tuve mucho éxito a la calle* (хотел «вышел на улицу») → *éxito* = успех! Выход — *salida*: *Salí a la calle*.
- ❌ *Estoy embarazada* как «я смущён(а)» → это «я беременна»! Смущён — *Estoy avergonzado/a*.
- ❌ *Actualmente* в значении «на самом деле» (англ. *actually*) → *actualmente* = в настоящее время; «на самом деле» — *en realidad / de hecho*.
- ❌ *Realicé que…* (англ. *I realized*) → ✅ *Me di cuenta de que…*
- ❌ путать паронимы: *infligir un castigo* (наложить наказание) vs *infringir la ley* (нарушить закон) — не наоборот.

## 🎯 Совет
- При сомнении проверяйте в **DRAE** все значения слова.
- Ложных друзей фиксируйте отдельным списком — они вызывают самые заметные и неловкие ошибки.
- Паронимы запоминайте парами, привязывая каждый к примеру-колокации (*infligir un daño* / *infringir una norma*).
