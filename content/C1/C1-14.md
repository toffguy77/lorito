> **Суть**
> Три источника лексических ловушек: многозначность (полисемия), близкие по форме слова (паронимы) и обманчивое сходство с русским/английским (ложные друзья).

## 🔀 Полисемия (одно слово — много значений)
| Слово | значения |
|-------|----------|
| **banco** | скамейка / банк / косяк рыб |
| **planta** | растение / этаж / завод / ступня |
| **cabo** | мыс / капрал / конец (al cabo de) |
| **dejar** | оставлять / позволять / бросать |
| **pasar** | проходить / случаться / передавать |

Контекст и колокация снимают неоднозначность (см. колокации).

## ⚖️ Паронимы (похожи по форме — разный смысл)
| Пара | различие |
|------|----------|
| *actual* (нынешний) / *actuar* (действовать) | |
| *especie* (вид) / *especia* (специя) | |
| *infligir* (наносить) / *infringir* (нарушать) | |
| *absolver* (оправдать) / *absorber* (впитывать) | |
| *aptitud* (способность) / *actitud* (отношение) | |
| *prever* (предвидеть) / *proveer* (снабжать) | |

## 🎭 Ложные друзья (с русским/английским)
| Испанское | НЕ значит | значит |
|-----------|-----------|--------|
| *éxito* | «выход» | успех |
| *fábrica* | «фабрика тканей?» | завод (любой) |
| *carta* | «карта» | письмо / меню |
| *embarazada* | «смущённая» | беременная |
| *sensible* | «sensible» (англ.) | чувствительный |
| *actualmente* | «actually» | в настоящее время |
| *realizar* | «realize» | осуществлять |
| *ropa* | «ropa» | одежда |

## 🎯 Совет
- При сомнении проверяйте в **DRAE** все значения.
- Ложных друзей фиксируйте отдельным списком — они вызывают самые заметные ошибки.
