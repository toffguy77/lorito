> **Суть**
> Колокация — устойчивое, «правильное» сочетание слов, которое носители используют автоматически. Знание колокаций — главный маркер естественности на C1.

## 🤝 Глагол + существительное (частотные)
| Колокация | (не дословно) |
|-----------|---------------|
| **tomar** una decisión | принять решение (не *hacer*) |
| **prestar** atención | уделять внимание |
| **cometer** un error / un delito | совершить ошибку/преступление |
| **correr** un riesgo | рисковать |
| **hacer** hincapié (en) | делать акцент |
| **poner** en marcha | запустить |
| **dar** un paseo / una vuelta | прогуляться |
| **sacar** una conclusión | сделать вывод |
| **guardar** silencio / un secreto | хранить молчание/секрет |
| **entablar** una conversación | завязать беседу |

## 🎨 Прилагательное + существительное (интенсификаторы)
- *un error **garrafal*** (грубая ошибка), *un fracaso **estrepitoso***, *una diferencia **abismal***, *un **rotundo** éxito*, *lluvia **torrencial***, *un **profundo** conocimiento*.

## 🔧 Наречие + прилагательное / глагол
- *profundamente dormido, firmemente convencido, rotundamente falso, llover a cántaros*.

## 🧱 Существительное + предлог + существительное
- *un rayo de esperanza, un ataque de risa, una pizca de sal, un cúmulo de circunstancias*.

## 🎯 Зачем учить
Грамматически верное, но «неколокационное» сочетание звучит чуждо:
- ❌ *hacer una decisión* → ✅ *tomar una decisión*.
- ❌ *fuerte error* → ✅ *error garrafal*.

## 💡 Совет
Записывайте новые слова **в составе колокации**, а не изолированно: не *decisión*, а *tomar una decisión*.
