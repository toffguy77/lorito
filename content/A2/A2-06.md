> **Суть**
> Оба — завершённые действия. Различие — в **периоде времени**: perfecto для незавершённого периода (включающего «сейчас»), indefinido — для законченного.

## ⚖️ Сравнение (норма Испании)

| | **Perfecto** (he hablado) | **Indefinido** (hablé) |
|--|---------------------------|------------------------|
| Период | связан с настоящим / не закончен | закончен, оторван от настоящего |
| Маркеры | hoy, esta semana, este año, ya, todavía, alguna vez, nunca | ayer, anoche, el lunes, en 2010, hace un mes |
| Пример | *Hoy **he comido** paella.* | *Ayer **comí** paella.* |
| Пример | *Este año **he viajado** mucho.* | *El año pasado **viajé** mucho.* |

## 🌎 Региональное различие
- **Испания:** соблюдает контраст (perfecto для «сегодня/недавно»).
- **Латинская Америка:** чаще везде **indefinido**: *Hoy comí* вместо *Hoy he comido*.

## 🎯 Подсказка
Спросите себя: период действия **ещё длится**? → perfecto. **Уже закрыт**? → indefinido.

## ⚠️ Частая ошибка
❌ *Ayer he comido* → ✅ *Ayer comí* (вчера — закрытый период).
