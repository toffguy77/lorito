> **Суть**
> Многие частотные глаголы в indefinido неправильны: меняется корень, а окончания особые (**без ударения** на окончании: -e, -iste, -o, -imos, -isteis, -ieron).

## 🧱 Группа с особым корнем (verbos pretéritos fuertes)
Окончания: **-e, -iste, -o, -imos, -isteis, -ieron** (у *-j-* корней: -**eron**).

| Инфинитив | корень | yo / él |
|-----------|--------|---------|
| tener | tuv- | tuve / tuvo |
| estar | estuv- | estuve / estuvo |
| poder | pud- | pude / pudo |
| poner | pus- | puse / puso |
| saber | sup- | supe / supo |
| hacer | hic- (hizo!) | hice / hizo |
| querer | quis- | quise / quiso |
| venir | vin- | vine / vino |
| decir | dij- | dije / dijo (dijeron) |
| traer | traj- | traje / trajo (trajeron) |

## 🚶 ser и ir — совпадают!
*fui, fuiste, fue, fuimos, fuisteis, fueron* — у обоих одинаково. Различаются по контексту:
- ***Fui** médico* (был врачом — ser). / ***Fui** a Madrid* (поехал — ir).

## 🎁 dar — особый
*di, diste, dio, dimos, disteis, dieron*.

## 🔁 Изменение корня в 3-м лице (-ir с e→i, o→u)
- *pedir → pidió, pidieron*; *dormir → durmió, durmieron*; *preferir → prefirió*.

## ⚠️ Запомнить
*hacer* → *él **hizo*** (c→z перед o ради звука). У *decir/traer/-ducir* в 3-м лице мн.ч. — **-jeron** (не -jieron): *dijeron, trajeron, condujeron*.
