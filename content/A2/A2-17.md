> **Суть**
> Три типа сравнения: больше, меньше, одинаково.

## 📐 Три модели

| Тип | формула | пример |
|-----|---------|--------|
| **больше** | **más** + прил./нареч./сущ. + **que** | *Soy más alto que tú.* |
| **меньше** | **menos** + … + **que** | *Es menos caro que el otro.* |
| **одинаково** | **tan** + прил./нареч. + **como** | *Soy tan alto como él.* |
| **одинаково (сущ.)** | **tanto/a/os/as** + сущ. + **como** | *Tengo tantos libros como tú.* |

## 🎯 Примеры
- *Madrid es **más** grande **que** Sevilla.*
- *Hoy trabajo **menos que** ayer.*
- *Ella habla **tan** bien **como** un nativo.*
- *No tengo **tanto** dinero **como** antes.*

## 🔢 Сравнение с числами → de (не que)
- *Hay **más de** cien personas.* — Больше ста человек.
- *Cuesta **menos de** diez euros.*

## 🔑 tan vs tanto
- **tan** + прилагательное/наречие: *tan alto, tan rápido*.
- **tanto/a/os/as** + существительное (согласуется): *tanta agua, tantos amigos*.

## ⚠️ Неправильные формы
*bueno → mejor*, *malo → peor*, *grande → mayor*, *pequeño → menor* — НЕ *más bueno*. См. Неправильные сравнения (mejor, peor).
