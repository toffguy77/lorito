> **Суть**
> Оба ≈ «для/за/через», но различаются по смыслу. Грубо: **para** смотрит вперёд (цель, адресат), **por** смотрит назад/вокруг (причина, способ).

## 🎯 PARA — цель, назначение, срок

| значение | пример |
|----------|--------|
| цель / «чтобы» | *Estudio **para** aprobar.* |
| адресат | *Este regalo es **para** ti.* |
| срок (к моменту) | *Es **para** el lunes.* |
| назначение/мнение | *Una taza **para** café. **Para** mí, es fácil.* |
| направление | *Salgo **para** Madrid.* |

## 🔄 POR — причина, способ, обмен, время

| значение | пример |
|----------|--------|
| причина / «из-за» | *Lo hago **por** ti. Gracias **por** todo.* |
| способ/средство | *Hablamos **por** teléfono. **Por** correo.* |
| обмен / цена | *Lo compré **por** 20 euros.* |
| место «через» | *Paseo **por** el parque.* |
| время суток / период | ***Por** la mañana. **Por** dos semanas.* |

## ⚖️ Минимальные пары
- *Estudio español **para** mi trabajo* (цель) vs *…**por** mi trabajo* (из-за работы).
- *Lo hago **para** ti* (тебе предназначено) vs *Lo hago **por** ti* (ради тебя).

## 🧱 Устойчивые выражения
- **por:** *por favor, por fin, por eso, por ejemplo, por supuesto, por la mañana*.
- **para:** *para siempre, para nada, ¿para qué?*

## ⚠️ Подсказка
**¿para qué?** = «зачем/с какой целью?»; **¿por qué?** = «почему/по какой причине?».
